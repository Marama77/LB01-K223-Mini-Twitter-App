# LB01 üK223 Mini-Twitter-App

Full-stack "mini-twitter" application built with **TypeScript**, **Node.js (Express)** and **MariaDB**, fully orchestrated with **Docker**.

# Tech stack
- **Frontend:** HTML/CSS/TypeScript
- **Backend:** Node.js + Express + TypeScript
- **Database:** MariaDB
- **Orchestration:** Docker Compose
